About This Project

Rajprabha Garden Villas is a thoughtfully planned villa township in Nashik's prestigious Gangapur Road corridor. Each row house features a private garden, ample living space, and premium finishes. The gated community provides a jogging track, clubhouse, temple, and round-the-clock security.
